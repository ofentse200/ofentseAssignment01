# Student Management System

A comprehensive Java-based Student Management System demonstrating object-oriented programming principles, design patterns, and SOLID principles. This system efficiently manages different types of students (Undergraduate, Graduate, and Part-Time) with flexible functionality for enrollment, tuition calculation, and reporting.

## Project Overview

This Student Management System is designed to handle multiple types of students with different enrollment patterns and tuition calculation methods. The system is built with extensibility and maintainability in mind, adhering to industry-standard design patterns and SOLID principles.

**Key Technologies:**
- Java 8+
- Object-Oriented Programming (OOP)
- Design Patterns (Factory, Builder, Service Layer)
- SOLID Principles

## Features

✅ **Student Management**
- Create undergraduate, graduate, and part-time students
- Add/remove students from the system
- Search students by ID or department
- Update student information

✅ **Tuition Calculation**
- Calculate individual student tuition
- Calculate total tuition across all students
- Calculate average tuition per student
- Different tuition calculation methods for each student type

✅ **Reporting & Analytics**
- Display all students with detailed information
- Filter students by department
- Get statistics on student types and counts
- Generate comprehensive student reports

✅ **Data Management**
- Support for multiple student types
- Flexible builder pattern for easy student creation
- Proper encapsulation and data hiding
